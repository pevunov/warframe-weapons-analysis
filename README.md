# Warframe Weapons Analysis

Проект по анализу оружия Warframe с использованием Python (pandas, numpy, matplotlib, seaborn).  
Включает:  
- Загрузка и очистка данных  
- Статистический анализ  
- Визуализации (гистограммы, столбчатые диаграммы, boxplot, heatmap, кольцевая диаграмма)  
- Группировки и агрегаты
